# BeatFlow

Projeto mobile em Expo/React Native para simular um aplicativo de streaming de música.

## Como rodar

```bash
npm install
npx expo start
```

Depois, use:
- `w` para abrir na web;
- `a` para Android, se o emulador estiver configurado;
- QR Code com Expo Go no celular.

## Telas obrigatórias

- Login
- Cadastro de usuário
- Tela inicial
- Favoritos
- Tocando música

## Dados simulados

As músicas estão em `src/data/musicas.ts`. O app não usa banco de dados: tudo é mockado para fins de teste.
